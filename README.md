# template-gulp

開発用 gulpテンプレート

---

## Requirements

- Node.js
- Yarn
- Gulp

## Install

### Step.1

`config.js`を開き、使用するタスクを指定（Boolean）

### Step.2

```sh
$ yarn setup # タスクの差分モジュールを自動インストール
```

## Usage

### watch

```sh
$ yarn start # or gulp
```

### build

```sh
$ yarn build # or gulp build
```
